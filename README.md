# 下载依赖

下载代码格式化工具

```bash
npm istall
```

下载后端依赖

```bash
cd serve
npm install
```

下载前端依赖

```bash
cd web
npm install
```

# 启动项目

这里包含启动前端与后端的指令

## 启动后端

运行以下代码启动后端

```bash
cd serve
npm run dev
```

构建指令

```bash
npm run build
```

运行指令

```bash
npm run serve
```

## 启动前端

构建指令

```bash
cd web
npm run build
```

运行、测试指令
```bash
npm run dev
```
